#virtual_assistant (NLP)
Inteligentny asystent wykorzystujący przetwarzanie języka naturalnego.
Eksperymentalny projekt asystenta głosowego/tekstowego, badający możliwości integracji sztucznej inteligencji z codziennymi zadaniami.

Kluczowe funkcjonalności:
  NLP Processing: Wykorzystanie bibliotek do analizy i rozumienia intencji użytkownika.
  Speech-to-Text / Text-to-Speech: Integracja modułów komunikacji głosowej.
  Task Automation: Możliwość wykonywania prostych operacji systemowych (uruchamianie aplikacji, sprawdzanie pogody itp.) poprzez komendy naturalne.
